# WPF × 18 Tage Tracker

Persönlicher Fitness & Ernährungs-Tracker für das Wolfs Protein Fasten Programm.

## Auf iPhone installieren

1. Diese Seite in **Safari** öffnen
2. Unten auf das **Teilen-Symbol** tippen (Quadrat mit Pfeil ↑)
3. **"Zum Home-Bildschirm"** wählen
4. **"Hinzufügen"** tippen

Fertig — App-Icon erscheint auf dem Homescreen.

## Features
- 18-Tage Trainingsplan (Studio + Hotel-Einheiten)
- WPF Ernährungsplan mit Tagesprotokoll
- Ernährungs-Logging mit Kalorien & Protein
- Kalenderansicht Mai/Juni 2026
- Offline-fähig (PWA)
- Daten bleiben gespeichert (localStorage)
